# BMW GraphQL API

BMW provides connected vehicle APIs through the BMW Connected Drive platform. The API covers vehicle status, remote services (locking, horn, lights), navigation destinations, charging management for EVs, trip data, and vehicle location.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from BMW REST API
**Documentation:** https://aos-portal.bmwgroup.com/basic/ui/

- Reference: https://aos-portal.bmwgroup.com/basic/ui/#/detail/vehicle-identification
- Schema: https://github.com/api-evangelist/bmw
